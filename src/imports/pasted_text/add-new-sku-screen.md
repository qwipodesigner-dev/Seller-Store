Design a clean, modern, and intuitive **“Add New SKU” screen** for a B2B Seller App.

The screen should be optimized for distributors and sellers to quickly create product listings. Keep the layout structured, visually grouped, and easy to scan. Use cards or sections with clear headings.

---

## 🧩 SCREEN LAYOUT

Create a **single scrollable screen** divided into the following sections:

---

## 🟢 1. Basic Product Details

Fields:

* Product Name
* Category (dropdown)
* Brand Name
* SKU Code
* Short Description
* Long Description

---

## 🟢 2. Product Images

Fields:

* Upload Images (multiple)
* Image preview thumbnails

---

## 🟢 3. Pricing

Fields:

* Selling Price
* MRP (Maximum Retail Price)
* Currency (default INR)

---

## 🟢 4. Inventory & Packaging

Fields:

* Available Quantity
* Maximum Order Quantity
* Unit Type (unit / kg / litre / etc.)
* Pack Size (e.g., 1L x 12, 5kg x 4)
* Packaging Type (single / case / bundle)

---

## 🟢 5. Fulfillment Options

Fields:

* Fulfillment Type (toggle or multi-select):

  * Delivery
  * Self Pickup
  * Both

* Store / Warehouse Selection

---

## 🟢 6. Shipping & SLA

Fields:

* Time to Ship (input with unit: mins / hours / days)

---

## 🟢 7. Return & Cancellation

Fields:

* Returnable (toggle)
* Cancellable (toggle)
* Return Pickup Available (toggle)

---

## 🟢 8. Payment Options

Fields:

* Cash on Delivery Available (toggle)

---

## 🟢 9. Compliance Information

Fields:

* Manufacturer / Packer Name
* Manufacturer Address
* Generic Product Name
* Month & Year of Manufacture / Packing

---

## 🟢 10. Customer Support

Fields:

* Support Name / Organization
* Email ID
* Phone Number

---

## 🟢 11. Origin Details

Fields:

* Country of Origin

---

## 🟢 12. Additional Details (Optional Section)

Fields:

* Weight
* Volume
* Variant (size / type / flavor)
* Tags

---

## 🎨 DESIGN STYLE

* Use a **card-based layout**
* Clear section headers
* Minimal, modern UI (similar to Shopify / Flipkart Seller)
* Use toggles, dropdowns, and input fields appropriately
* Keep spacing clean and readable
* Add primary CTA buttons at bottom:

  * Save Draft
  * Publish SKU

---

## 🧭 UX FLOW

* Top: Page title “Add New SKU”
* Middle: Structured sections (as above)
* Bottom: Action buttons (Save / Publish)

---

The design should feel **fast, structured, and business-friendly**, enabling sellers to quickly add products without confusion.
